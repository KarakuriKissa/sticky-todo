# プライバシーポリシー — StickyTodo

最終更新: 2026-05-07

StickyTodo（以下「本アプリ」）は利用者のプライバシーを尊重します。本ポリシーで本アプリが
収集・送信・保存するデータの一切を明らかにします。

## 1. 収集・送信しない情報
本アプリは下記の情報を **一切** 収集・送信しません。

- 氏名・メールアドレス・電話番号などの個人を識別する情報
- アプリで作成したリスト名、タスク本文、コメント、添付メモ
- 操作ログ、利用統計、アクセス時刻、画面録画、スクリーンショット
- IP アドレスや位置情報
- アクセスしたウェブサイトやファイル一覧

## 2. ローカル保存される情報
本アプリは下記のデータを利用者の PC 内のみに保存します。サーバーへの送信は一切ありません。

| 保存先 | 内容 |
|---|---|
| `%APPDATA%\com.stickytodo.app\sticky-todo.db` (SQLite) | リスト・タスク・カテゴリ・ステータス・担当者・設定 |
| ブラウザ localStorage (本アプリの WebView) | 上記の冗長バックアップ、設定、最後に表示したビルド番号 |

利用者は **設定 > 詳細設定 > データベースを削除** でいつでもこれら全てを完全に消去できます。

## 3. アップデート確認のための通信
本アプリは起動時および「設定 > ヘルプ」で「最新ビルドを確認」を押下した時に、
GitHub の **公開 API** (`https://api.github.com`) に対し下記の HTTPS リクエストを送ります。

- `GET /repos/TomTomYukkie/sticky-todo/actions/workflows/build.yml/runs?per_page=1&status=success`

このリクエストには:

- ✅ クライアント側（本アプリ）から個人情報や利用データは含まれません
- ✅ GitHub のサーバーには標準的なアクセスログ（IP・User-Agent 等）が残ります（GitHub のプライバシーポリシーが適用されます）
- ✅ アップデート通知をしたくない場合: ネットワーク接続を切断するか、ファイアウォールで `api.github.com` への発信をブロックしてください

## 4. URL を開く処理
タスク本文中の `https://...` をクリックすると、本アプリは Tauri shell プラグイン経由で
**OS 既定のブラウザ** を起動して当該 URL を開きます。本アプリが直接サーバーと通信することはありません。

## 5. データのエクスポート / インポート
利用者は **設定 > 詳細設定 > エクスポート** で SQLite ファイルそのものを任意の場所に保存できます。
ファイルは利用者の責任で管理してください。第三者に渡す場合、リスト・タスクの内容はすべて含まれます。

## 6. 子どものプライバシー
本アプリは年齢に依らず利用可能ですが、特に 13 歳未満の児童の個人情報を **意図的に取得することはありません**。

## 7. 改訂
本ポリシーは予告なく変更される場合があります。変更があった場合は本ファイルの「最終更新」日付
および GitHub のコミット履歴で確認できます。
